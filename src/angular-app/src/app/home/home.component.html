Bookshelf
A social media for books This project was built in microservices architecture, using C# / .NET for backend and Angular / Material for frontend. Each microservice uses MongoDB as its database, and inter-service communication is achieved using RabbitMQ. The frontend and backend communicate via Ocelot API gateway.
